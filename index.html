About Me

I have 16 years experience as a middle school science teacher; I decided to formally pursue web development in the past year in order to use code as a medium to reach new audiences.

I am a web developer and digital designer with 10 years experience in editing, motion graphics and educational curriculum design.

I am passionate about challenging myself and about finding innovative ways to communicate and solve problems through coding and design.

Skills

Development

I have created web applications that use APIs such as Flicker and iTunes to create searchable songs and pictures based on the player’s selection. These were built using Ruby on Rails, Javascript, jQuery, HTML5, CSS3, Bootstrap and Bass CSS.

Editing

I have edited short films, promotional videos and long form videos using Adobe Premier, Final Cut Pro and After Effects.

Design

I believe that design should be elegant, simple, and effective in communicating the client’s message. My design work ranges from promotional advertisements to film motion graphics to an educational science website.

Collaboration

I enjoy collaborative work and value the synergy that comes from working in a team. I am always open to collaboration.

App Development
Flickrpics

Flickrpics

Project Description

A simple one page app that was built using the Flickr API that helps you keep track of the latest posted pics based on categories, friends id and friends favorites. Built using the Flickr API, JavaScript, jQuery, Ajax, JSON, HTML5, CSS3 and Bootstrap. The app helps you keep your finger on the pulse of the latest picture trends on flickr. You can search for the most recent posts on flickr by category, friends or your friend’s favorites.

Type in a search term to find the most recent photos that have been posted on Flickr.If you are too busy to join Flickr but want to stay connected to your friends by viewing their recent pictures type in your friend’s username and see what their most recent posts are. Keep up-to-date on your friends’ favorite pictures too by tying your friend’s id to find their favorite photos.

App Development
Trackmaster

Project Title

Project Description

Music quiz web application that uses the iTunes API to create a fun listening tests that you can play with your family and friends. There various levels of any artist of the players choosing, complete with arcade scores and leaderboards. Built using iTunes API, Ruby on Rails, Javascript, HTML, CSS. The app lets you search an artist and load a quiz of 5 songs performed by the artists.

You can play each song and there are four possible answers to choose from and you are given 30 seconds to answer each question. You can get higher scores if you are able to identify the songs and choose the right answer. There is also a signup option if you want you scores saved and compared to other users

Wasistdas

Project Description

A simple one page application that allows you to post photos and to respond to the photos you posted as well as ones posted by other members. Built with Ruby on Rails, Javascript, Ajax, JSON HTML and Bass CSS. You can signup and load a picture using the picture’s url and other members can vote the picture up or down. You can also comment on the picture and users can also reply to the comment.
